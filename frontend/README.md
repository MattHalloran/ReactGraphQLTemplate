# NLNWebsite

NLNWebsite is a new, modern website created for New Life Nursery, Inc. This code serves to set an example for anyone looking to start a React/Flask website.

## Start React Frontend

```terminal
cd nln-website

npm start
```

## Start Flask Backend

```terminal
cd nln-website/api

flask run
```
